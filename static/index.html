MarkItDown 사용 설명서

💡 AI 없이 즉시 무료 변환 (로컬 변환)

기본 마크다운 변환 엔진을 활용해 OCI 서버 자체적으로 파일을 해독하여 즉시 마크다운으로 변환합니다.

대상 파일 변환 형태
Word (`.docx`) 본문 본편 구조, 제목 스타일, 표 자동 복원
Excel (`.xlsx`) 스프레드시트 시트 데이터를 마크다운 표로 변환
PPTX (`.pptx`) 슬라이드별 텍스트 박스를 구조화된 문서로 추출
PDF (텍스트형) 드래그가 가능한 일반 PDF의 텍스트와 데이터 복원
텍스트 (`.txt`, `.csv`, `.json`) 인코딩 정돈 및 마크다운 원문 구성

✨ AI(Gemini) 연동 변환 (고급 분석)

Google Gemini API Key를 입력하여 활성화하면 시각/청각 자료를 고도의 AI 모델로 해석합니다.

  • 이미지 분석 (OCR / 설명)

    이미지(`PNG`, `JPG`) 속 글자를 정밀 분석하여 텍스트로 추출하며, 아키텍처 다이어그램이나 복잡한 그래프를 분석하여 설명글을 마크다운 문서로 작성해 줍니다.

  • 오디오 음성 전사 (Speech-to-Text)

    통화 녹음 및 대화 파일(`.m4a`, `.mp3` 등)의 음성을 분석하여 대화 형식(녹취록)으로 전사하고, 문서 최하단에 한글 요약(Summary)을 자동 추가합니다.

  • 오류 대처 팁 (Rate Limit/Quota 429/503)

    구글 API 호출 도중 서버 트래픽이 몰리거나 한도 초과 시 429/503 에러가 발생할 수 있습니다. 이 경우 사용할 AI 모델 드롭다운을 `gemini-3.1-flash-lite`로 설정해 우회해 보세요.
